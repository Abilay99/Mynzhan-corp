 Әйгілі Microsoft компаниясының киберқауіпсіздік жұмыстары бойынша сарапшысы Крис Джексон Internet Explorer браузерін пайдалану қауіпті деп хабарлады. Сарапшының жазбасы Windows IT Pro ресми блогында жарияланған. Бұл хабардың тарауына браузердің 2015 жылдан бері бір де бір рет жаңармауы әсер еткен. Яғни, браузер жаңа веб-стандартты көрмейді. Крис Джексон өз жазбасында браузер осы күйі жаңармайтын болса, алдағы уақытта қолданушыларынан айырылып қалатынын жазды. Сонымен қатар, Internet Exlprer-ді енді браузер деп атамау керектігін алға тартып қолданушыларды оны пайдаланудан бас тартуға шақырды. Компания ескі браузердің баламасы ретінде Microsoft Edge браузерін қолдануды ұсынып отыр. Жақын уақытта бқл браузер Windows 10, Windows 8, Windows 7, Mac, Linux бағдарламаларында қолжетімді болады. Джексонның болжауы бойынша, Edge веб-стандарттарды өзгерту бойынша бірінші орынға шығады.
