Естеріңізде болса Windows операциялық жүйесі алғашында Windows 95, Windows 98 деп аталатын. Ал, қазіргі Windows 8 жүйесі де бірден Windows 10-ға өтіп кетті. Мұның қандай да бір себебі бар ма? Әрине, бар екен. Оның басты себебі, Windows 95, 98 жүйесіне арнап жазған кейбір бұйрықтар жаңа Windows 9 жүйесіндегі кейбір бұйрықтарды өзгертіп, қателіктерге ұрындыруы мүмкін екен. Мысалы, бұрынғы Windows мынадай бұйрықтармен қозғалса: { /*Windows 95, Windows 98 */ } else{ }, енді, оны Windows 9-ға өзгерту үшін if ( version.StartsWith("Windows 9") ) деген сияқты бұйрықтарды атқару керек. Сондықтан, Microsoft қолайсыз жағдайдан шығудың амалын оңай ойлап тапқан. Ол  амал Windows-ты бірден 10-шы нұсқада шығару болды. Сурет: windowsphone-italia.com
