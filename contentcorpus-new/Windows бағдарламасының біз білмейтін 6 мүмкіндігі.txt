 Windows бағдарламасындағы көптің білгірі біле бермейтін мынадай командалар бар: 1. Win-X: сіздің жаңа жылдам көмекшіңіз Windows-тің көптеген қолданушылары бұл әмбебап шорткат туралы өте аз біледі. Ол Windows Mobility Center бақылау панелі (тек Windows 7 (және Vista) платформаларында ғана) арқылы іске қосылады. Mobility Center басқару орталығы мобильді компьютерге қатысты бағдарламалардың барлығымен желі арқылы байланыста болады. Егер сіз сыртқы дисплей немесе Windows Sync Center платформалық нұсқасын қолданатын болсаңыз, сізге Mobility Center арқылы кез-келген батырма қол жетімді болады. Ал Windows 8 нұсқасындағы Win-X комбинациясы біршама өзгерген. Енді ол операциялық жүйенің өзге компоненттерінің шорткаттарын да көрсететін болған. 2. Әр басылған батырмаңызды сақтап алыңыз Енді  Windows бағдарламасының құрастырушылары Problem Steps Recorder деген атауға ие құрылғыны жасап шығарды. Сіз оны Windows 7 нұсқасында "Қосу" (Пуск) батырмасындағы «Record steps to reproduce a problem» деген команда арқылы, ал Windows 8-де «Steps Recorder» деген атауды енгізу арқылы таба аласыз. Аталған құрылғы ашылған бойда, мониторда кішкене диалог беті ашылады. Сосын бір ғана "Start Record" батырмасын бассаңыз болғаны. Содан кейін, Windows сіздің тышқанмен жасаған барлық операциялық жұмыстарыңызды жазып отырады. Сосын жинақталған ақпарат автоматты түрде ZIP-архивте HTML-құжат болып сақталады. Сақталған құжатта әрбір әрекеттің түсіндірмелері мен скриншоттары қамтылған болады. 3. Тапсырмалар панеліндегі Shift-Click таңдалған қосымшаның көшірмесін жасайды Арнайы Shift батырмасын басып тұрып, тапсырмалар панеліндегі кез-келген қосымшаны басыңыз. Сосын жүйе екінші оның көшірмесін ашады. Аталған операцияның ресми жолы тышқанның оң батырмасын басу арқылы жасалынады. Бұл Windows операциялық жүйесінің барлық нұсқасында бірдей орындалады. 4. Скриншоттарды бірден өңдеу Windows платформасындағы Print Screen қолданысы қарапайым командалар арасында өте қолайлысы деуге болады. Ал Windows 7 Snipping Tool жаңа құрылғысы сізге суретті алып, оны кесіп, көлемін өзгерте отырып, қайnа сақтау жұмыстарын бірден орындауға мүмкіндік береді. Сонымен қатар, Windows 8 нұсқасындағы Snipping Tool алдынғы нұсқасынан ерекше артықшылығы бар. PrtSc батырмасын басудан бұрын Win-PrtSc батырмасын басыңыз. Бұл сізге толық экранның скриншотын жасайтын болады. Сосын автоматты түрде суреттер галереясында сақталынады. 5. Right-click - пошта арқылы файл жіберуге арналған команда Егер сіз Outlook, Windows 7 немесе 8 платформаларының оффлайн пошта қызметін қолданатын болсаңыз, сіз ол арқылы сақталған файлдарды еш қиындықсыз жібере аласыз. Алдымен файлды бір рет басып, сосын Send to > Mail recipient командаларын басыңыз. Сіздің пошта қызметіңіз таңдалған файл енген жаңа хабарлама жіберу бетін ашады. Сосын қажетті аралықтарды толытырып, "жіберу" батырмасын басу ғана қалады. 6. Бэкап (Қосымша көшіру функциясы) құру үшін құжаттар тарихын қолдану Windows 8 платформасында бэкап жасауға арналған File History деген атауға ие қарапайым қызмет түрі қосылған. Оны қолдану үшін сыртқы жинақтағышын іске қосасыз. Сосын Windows автоматты түрде барлық файлдарды сақтап отыратын болады. Егер абайсызда маңызды құжатты немесе бағдарлама файлын өшіріп алатын болсаңыз, оны қайта қалпына келтіруге мүмкіндігіңіз болады. Windows іздеу қызметінен File History қызметін тауып, іске қосасыз. Ал өзге файлдарды қайта қалпына келтіру функциялары тышқанның оң батырмасын басып, «Restore your files with File History» командасын басу арқылы жүзеге асырылады. Дайындаған: Жолдас Өрісбаев
