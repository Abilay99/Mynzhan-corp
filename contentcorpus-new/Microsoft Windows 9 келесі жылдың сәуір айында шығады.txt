Microsoft корпорациясының мамандары әзірге "Threshold" атауына ие жаңа операциялық жүйені (ОЖ) жасаумен айналысып жатыр. Шетел баспасөзі берген мәлімдемеге сүйенер болсақ, ОЖ 2015 жылдың сәуір айында қолданысқа шығады екен. Сонымен қатар, жаңа қызмет түрі Windows 9 атауына ие болып, Microsoft корпорациясының барлық қолданушыларға қолайлы болған Windows 7 ОЖ-не бағытталып жасалынады деп күтілуде. Жаңа нұсқа жарыққа шыққаннан кейін алдыңғы Windows 8 нұсқасынан барынша аластауға тырысатын болады. Себебі қазіргі соңғы модель онша сәтті қолданылып жүрген жоқ. Net Applications статистикалық қызметінің ақпаратына орай, 2013 жылдың қараша айына дейін Windows 8 және Windows 8.1 ОЖ-нің нарықтағы жалпы алған пайызы 9,3% болыпты. Бұл Microsoft корпорациясының тарихындағы ең төмен көрсеткіш болып табылады.
